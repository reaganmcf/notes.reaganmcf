Categorical, Numerical, and Ordinal Variables

  • CAT: Grade like A, B, C, D
  • NUM: Numerical: SCORE: like 89.64
  • ORD: Ordinal ordered categorical: D < C < B < A

Which plot to use

  • It all depends on the variables
    • NUM x NUM = Scatter
    • CAT x CAT = Mosaic
    • CAT x NUM = Box
    • NUM = Bar plot, Histogram
    • CAT = Bargraph
